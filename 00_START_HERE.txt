⚡ THE THREE COMMANDS YOU NEED:
================================================================================

Everything works in 3 commands:

COMMAND 1 (5 minutes):
    bash setup.sh
    
    This installs dependencies and creates the project structure.

COMMAND 2 (10-20 minutes):
    make data
    
    This prepares your data:
    - Downloads videos (if using URLs)
    - Extracts frames
    - Extracts keypoints
    - Creates train/val/test splits

COMMAND 3 (12+ hours):
    make ssl && make train
    
    This trains your model:
    - SSL pretraining (4-8 hours)
    - Supervised fine-tuning (1-2 hours)
    - Saves best model to models/best_model.pt

That's it! Your trained model is ready.
